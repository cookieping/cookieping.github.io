這個這個網站是最近開始學習寫網頁，慢慢摸索寫出來的。
其中排版的樣子參考了台大孔令傑教授的個人網站， 而製作的過程有使用 Bootstrap 的樣式。

雖然目前的網頁開發能力仍有許多不足，但在寫網頁的過程中可以感覺到自己慢慢進步、慢慢發現某個地方可以怎麼寫會更好。我在這個過程中發現許多樂趣，也很期待能在這個領域探索更多。

目前會的＆學習中的語言：
C / C++ (資料結構 / 演算法 / SFML)
Python (numpy / pandas / matplotlib / sklearn)
HTML / CSS / Javascript (基礎使用、初學)
Maya (動畫製作)

以下是一些程式開發的紀錄：

  • 機器學習概論期末專案：預測番茄甜度
    以「如何從番茄外觀判斷其甜度」為題，用Python進行資料視覺化、sklearn模型預測、分析評估。我在小組中負責資料蒐集、資料視覺化、KNN方法的實作與分析。
  • 算法設計與分析期末專案：圖像修復
    基於《Kernel Regression for Image Processing and Reconstruction》論文，以C++實作核回歸算法與改良的濾波算法修復像素缺失圖像，我們組最終的修復結果為:除了圖像邊框外，肉眼判別該圖像幾乎無瑕疵。我在小組中負責核回歸算法的實現、濾波算法的實現與改良。
  • 電腦動畫與特效期末專案：短篇動畫 DUNGEON
    以MAYA製作近四分鐘的短篇動畫。我在小組中負責動畫的01:47~03:48部分製作。
  • 物件導向與資料結構專案：DUNGEON GAME
    此個人專案，以C++與SFML製作一個具有完整遊戲體驗的類RPG遊戲。
  • 計算機概論期末專案：亞馬遜棋遊戲
    基於貪心算法，以C++開發可人工對戰、AI對戰的亞馬遜棋遊戲。此為個人專案，於全體學生之專案的AI對戰中，獲得前三分之一的成績。
